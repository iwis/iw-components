Form

Form with all field types supported by all browsers. The fields new in HTML5 are marked. The form presents different types of constraints, for example required or min value. If the form contains incorrect fields, they will be marked in red after clicking Submit button. The form is responsive by default - it has a different layout on small and large screens. Additionally, the layout of the form can be customized in a CSS file thanks to the grid layout.
