Carousel with possibility to show many images at once.

All images in a carousel should have the same width (or be set to the same width in CSS).

Component isn't responsive. If it is used on mobile devices, set its width properly. Or use https://kenwheeler.github.io/slick/ .

based on: https://javascript.info/introduction-browser-events > Carousel

example usage
-------------
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">

  <script type="module" src="lib/iw-carousel-mul/iw-carousel-mul.js"></script>

  <link  href="lib/iw-carousel-mul/iw-carousel-mul.css" rel="stylesheet">
  
  <style>
    /* Set the size of a single image and the total width of a carousel (to a multiply of the width of an image)*/
    .iw-carousel-mul1                                 { width:  520px; }
    .iw-carousel-mul1 .iw-carousel-mul__inner ul img  { width:  130px;
                                                        height: 130px; }
    
    .iw-carousel-mul2                                 { width:  300px; }
    .iw-carousel-mul2 .iw-carousel-mul__inner ul img  { width:  100px;
